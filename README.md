# 🤰 胎动记录

一个温馨美观的胎动记录应用，帮助准妈妈记录宝宝的每一次小动作。

## 功能特性

- ✨ 简单易用，点击即可记录胎动
- 📊 实时统计：今日胎动、本次记录、时长、平均每小时胎动数
- ⏱️ 会话管理：开始/结束/重置记录会话
- 📜 完整历史：查看所有历史记录
- 💾 自动保存：数据保存在浏览器本地
- 🎨 精美界面：温馨柔和的配色，适合孕妇使用
- ⌨️ 快捷键支持：空格键快速记录
- 📱 响应式设计：支持手机和平板

## 使用方法

1. 点击"开始记录"按钮
2. 每次感觉到胎动时，点击大按钮或按空格键
3. 点击"结束记录"完成本次记录
4. 查看统计数据和历史记录

## 为什么记录胎动？

胎动是宝宝健康的晴雨表。定期记录胎动有助于：

- 📈 了解宝宝的作息规律
- 🔍 及时发现异常情况
- 👩‍⚕️ 为产检提供参考数据
- 💕 增进与宝宝的互动

## 温馨提示

- 建议每天固定时间记录，如早中晚各一次
- 感觉10次胎动大约需要多久，平均应在1-2小时内
- 如果胎动明显减少或异常，请及时就医
- 宝宝最活跃的时段通常是晚上9-11点

## 技术栈

- 纯 HTML5 / CSS3 / JavaScript
- 使用 localStorage 进行本地数据存储
- 响应式设计，支持移动端
